# Description
Simple calendar app with tasks made according CQRS pattern

# Features
Functionality:
- Calendar view
- Tasks CRUD
- Login/Registration

Tecnologies:
backend
- c# .net 6 ( Asp.net core )
frontend
- HTML5, SCSS, TypeScript + JQuery

Database - PostgreSQL
